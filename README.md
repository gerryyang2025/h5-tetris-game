# 俄罗斯方块大作战 (Tetris Blitz)

一个经典的 H5 俄罗斯方块游戏，支持触屏和键盘操作，配有得分系统、排行榜、音效和霓虹风格视觉设计。

- **游戏名称**：俄罗斯方块大作战 (Tetris Blitz)
- **当前版本**：v1.0
- **游戏类型**：经典益智类 H5 游戏
- **核心玩法**：控制方块下落、旋转、消除得分
- **目标用户**：移动端和电脑端玩家

## 游戏特色

- **经典玩法**：7 种标准方块（I、O、T、S、Z、J、L），支持旋转、左右移动
- **等级系统**：每消除 10 行升级，速度逐渐加快
- **下一个预览**：显示即将出现的方块，提前规划布局
- **幽灵方块**：半透明显示当前方块下落位置
- **粒子特效**：消除行时的炫酷动画
- **音效系统**：Web Audio API 生成，消除、旋转、移动、落地音效
- **背景音乐**：经典 Tetris 背景音乐
- **排行榜**：本地存储 Top 10 成绩
- **计时系统**：游戏内实时显示用时
- **响应式设计**：完美适配手机和电脑
- **霓虹风格**：GitHub 风格深色主题，炫彩视觉效果

## 操作方式

### 电脑键盘

| 按键 | 功能 |
|------|------|
| `←` `→` | 左右移动 |
| `↑` | 顺时针旋转 |
| `↓` | 加速下落（软降） |
| `Space` | 快速落底（硬降） |
| `P` | 暂停 / 继续 |

### 手机触屏

| 手势 | 功能 |
|------|------|
| 左滑 | 向左移动 |
| 右滑 | 向右移动 |
| 上滑 | 旋转 |
| 下滑 | 加速下落 |
| 快速下滑（距离大 + 速度快） | 快速落底 |

## 得分规则

| 消除行数 | 得分 |
|----------|------|
| 单行 | 100 × 等级 |
| 双行 | 300 × 等级 |
| 三行 | 500 × 等级 |
| 四行（Tetris!） | 800 × 等级 |
| 快速落底 | 每格 2 分 |

## 等级系统

- 每消除 **10 行** 升级
- 初始下落速度：1000ms/格
- 最高等级速度：100ms/格
- 等级越高，方块下落越快，得分倍率越高

## 技术栈

- **HTML5 Canvas** — 游戏棋盘渲染
- **Vanilla JavaScript** — 游戏逻辑与渲染循环
- **Web Audio API** — 程序化生成音效
- **localStorage** — 排行榜数据持久化
- **无外部依赖** — 单文件即可运行
