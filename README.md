# 俄罗斯方块大作战 (Tetris Blitz)

一个经典的 H5 俄罗斯方块游戏，支持触屏和键盘操作，配有得分系统、排行榜、音效和霓虹风格视觉设计。

- **游戏名称**：俄罗斯方块大作战 (Tetris Blitz)
- **当前版本**：v1.1.1（掌机版）
- **游戏类型**：经典益智类 H5 游戏
- **核心玩法**：控制方块下落、旋转、消除得分
- **目标用户**：移动端和电脑端玩家

## 游戏特色

- **掌机风格界面**：复古掌机造型，深色渐变外壳
- **虚拟按键**：屏幕上的十字方向键和功能按钮，方便触屏操作
- **经典玩法**：7 种标准方块（I、O、T、S、Z、J、L），支持旋转、左右移动
- **等级系统**：每消除 10 行升级，速度逐渐加快
- **下一个预览**：显示即将出现的方块，提前规划布局
- **幽灵方块**：半透明显示当前方块下落位置
- **粒子特效**：消除行时的炫酷动画（代码中通过消行动画实现）
- **音效系统**：Web Audio API 程序化生成音效（移动、旋转、落地、消行、游戏结束）
- **背景音乐**：经典 Tetris 背景音乐（tetris.mp3），支持开关
- **排行榜**：本地存储 Top 10 成绩，含游戏次数、最高分、最佳用时统计
- **计时系统**：游戏内实时显示用时
- **响应式设计**：完美适配手机和电脑，移动端自动缩放
- **霓虹风格**：GitHub 风格深色主题，炫彩视觉效果

## 操作方式

### 电脑键盘

| 按键 | 功能 |
|------|------|
| `←` `→` | 左右移动 |
| `↑` | 加速下落（软降） |
| `↓` | 加速下落（软降） |
| `Z` | 逆时针旋转 |
| `X` / `Space` | 顺时针旋转 |
| `Enter` | 快速落底（硬降） |
| `P` / `Esc` | 暂停 / 继续 |

### 手机触屏

#### 虚拟按键
| 按键 | 功能 |
|------|------|
| 十字方向键 ←→ | 左右移动 |
| 十字方向键 ↓ | 加速下落 |
| 十字方向键 ↑ | 加速下落 |
| ↺ 按钮 | 逆时针旋转（CCW） |
| ↻ 按钮 | 顺时针旋转（CW） |
| A 按钮 | 快速落底（HARD DROP） |
| B 按钮 | 暂停 / 继续（PAUSE） |
| M 按钮 | 音乐开关（MUSIC） |

#### 触屏手势
| 手势 | 功能 |
|------|------|
| 点击暂停遮罩 | 继续游戏 |

## 得分规则

| 消除行数 | 得分 |
|----------|------|
| 单行 | 100 × 等级 |
| 双行 | 300 × 等级 |
| 三行 | 500 × 等级 |
| 四行（Tetris!） | 800 × 等级 |
| 快速落底 | 每格 2 分 |

## 等级系统

- 每消除 **10 行** 升级
- 初始下落速度：1000ms/格
- 最高等级速度：100ms/格
- 等级越高，方块下落越快，得分倍率越高

## 技术栈

- **HTML5** — 页面结构
- **CSS3** — 掌机造型样式、响应式布局、动画效果
- **Vanilla JavaScript** — 游戏逻辑与 DOM 渲染
- **Web Audio API** — 程序化生成音效和背景音乐播放
- **localStorage** — 排行榜数据持久化
- **无外部依赖** — 单文件即可运行

## 游戏界面

游戏采用掌机造型，GitHub 风格深色主题：

- 掌机外壳：深色渐变 (#2d333b → #1a1a2e → #0d1117)
- 屏幕区域：内阴影效果，显示游戏画面
- 虚拟按键：渐变色按钮，支持触屏操作
- 方块配色：7 种霓虹色（I=青、O=橙、T=紫、S=绿、Z=红、J=蓝、L=橙）
- 暂停遮罩：半透明黑色，点击可恢复游戏
- 帮助弹窗：内置玩法说明，无需跳转页面
